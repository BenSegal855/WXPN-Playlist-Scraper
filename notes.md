# WXPN Record Scraper
Start of online record: March 10, 2013
Request format: `https://origin.xpn.org/utils/playlist/json/YYYY-MM-DD.json`

Record format:
```json
{
	"artist": "string",
	"song": "string",
	"album": "string",
	"timeslice": "YYYY-MM-DD HH:mm:SS",
	"image": "URL",
	"streamPreview": "URL"
}[]
```
